# Économie HELYOS — La boucle

`v0.2` · Statut : **Stable (modèle)**

---

## Le moteur

HELYOS n'est pas financé pour produire des revenus ; il **produit des revenus pour se financer**. La finalité reste la recherche et l'augmentation des capacités humaines (ADN 13).

```
        ┌──────────────┐
        │  RECHERCHE   │◀───────────────────────┐
        │ (Labs/R&D)   │                         │
        └──────┬───────┘                         │
               │ génère                          │ refinance
               ▼                                 │
        ┌──────────────┐     ┌──────────────┐    │
        │  PRODUITS    │────▶│   REVENUS    │    │
        │ (Ventures)   │     │              │    │
        └──────────────┘     └──────┬───────┘    │
                                    │ constitue   │
                                    ▼             │
                             ┌──────────────┐     │
                             │  PATRIMOINE  │─────┘
                             │              │
                             └──────────────┘
```

> **Recherche → Produits → Revenus → Patrimoine → Recherche.**
> Chaque tour de boucle doit, idéalement, augmenter la capacité d'investissement du tour suivant.

## Les trois objectifs financiers (du Codex v0.1)

1. **Rémunérer durablement le fondateur** — la vie matérielle du Conservateur est un prérequis, pas une option.
2. **Constituer un patrimoine** — un capital qui rend la recherche indépendante des aléas.
3. **Financer massivement la R&D** — l'essentiel du surplus retourne dans la recherche.

## Étoile Polaire

> **Cible :** atteindre le point où **HELYOS finance sa propre évolution** — où les revenus des produits couvrent durablement la R&D et la rémunération, rendant le système auto-entretenu.

C'est le critère de succès de la [Vision](../00_VISION/00_Vision.md).

## Garde-fous

- **GR-7** : aucune activité financière (ex. module Trading) n'est jamais autonome ; **toujours** validation humaine.
- **Réinvestissement prioritaire** : le défaut est de refinancer la R&D, pas de distribuer.
- **Mesure** : chaque produit déclare sa **friction supprimée** (ADN 14) et son **revenu net** — pas de produit « vanité ».
