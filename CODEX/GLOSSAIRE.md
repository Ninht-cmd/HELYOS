# Glossaire HELYOS

`v0.2` · Le vocabulaire fait partie de la source de vérité. Un terme = un sens.

---

| Terme | Définition |
|-------|------------|
| **HELYOS** | *Helye + OS.* L'entreprise/institution qui construit un système d'exploitation de l'intelligence personnelle. |
| **Jarvis** | Le **produit** d'HELYOS : l'intelligence opérationnelle. = Kernel + Mémoire + Agents + Outils + Gouvernance + Observabilité. |
| **Conservateur** | Le fondateur. Propriétaire de la donnée, décideur final, gardien du Codex. Détient le veto inconditionnel. |
| **Codex** | La **source de vérité** versionnée. Prime sur le code, les modèles et les conversations. |
| **ADN** | Les 16 principes fondateurs non négociables. Niveau d'autorité le plus élevé. |
| **Kernel** | Le cœur exécutable de Jarvis : bus, gouvernance, mémoire, registre d'agents. |
| **EventBus** | Bus pub/sub, colonne vertébrale de communication entre composants. |
| **Agent** | Unité d'intelligence spécialisée, enregistrée, déclarant le niveau d'autonomie qu'elle requiert. |
| **Autonomie graduée (A0–A5)** | Échelle d'autonomie : A0 Lecture, A1 Préparation, A2 Exécution validée, A3 Faible risque, A4 Gestion contrôlée, A5 Stratégique. |
| **Règles d'Or (GR)** | Interdits absolus, valables à tout niveau (ex. pas de suppression sans backup). Codés et testés. |
| **PolicyEngine** | Composant qui décide `ALLOW` / `REQUIRE_VALIDATION` / `DENY` pour toute action. |
| **AuditLog** | Journal immuable de toutes les décisions de gouvernance. |
| **OODA** | Boucle Observe–Orient–Decide–Act. L'étape *Act* est gouvernée. |
| **Mode No-AI** | Chemin déterministe, sans appel à un modèle. Reproductible, auditable, résilient. |
| **RuView** | Module de *Wi-Fi sensing* : perception de présence/mouvement via signaux radio, sans caméra. |
| **ADR** | *Architecture Decision Record.* Décision technique tracée et immuable. |
| **RFC** | *Request For Comments.* Proposition structurante mise en discussion. |
| **Friction** | Toute perte de temps/énergie répétée. Une feature doit en supprimer une, **mesurable** (ADN 14). |
| **Boucle économique** | Recherche → Produits → Revenus → Patrimoine → Recherche. |
| **HELYOS OS / Labs / Research / Ventures** | Les branches : la fondation / les prototypes / la R&D long terme / les produits monétisables. |
| **Local-first** | Le cœur fonctionne sans Internet ; le cloud est opt-in, jamais requis. |
